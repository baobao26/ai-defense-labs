# ai-defense-labs

Umbrella repo grouping this workspace's blue-team / detection-engineering projects.

## Projects

- [mcp-hayabusa](https://github.com/baobao26/mcp-hayabusa) (submodule) — MCP server wrapping Hayabusa for Sysmon/EVTX analysis.
- [sysmon-parser](https://github.com/baobao26/sysmon-parser) (submodule) — Sysmon log parsing tools.
- [siem-queries](https://github.com/baobao26/siem-queries) (submodule) — Saved SIEM/Splunk queries and investigation notes.

Clone with submodules:

```
git clone --recurse-submodules https://github.com/baobao26/ai-defense-labs.git
```
